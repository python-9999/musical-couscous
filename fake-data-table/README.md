# Fake Data Table

Мини-скрипт на Python, который генерирует случайные имена и возраста и выводит их в таблицу с помощью библиотеки "rich".

## О проекте

- Выводит таблицу с двумя столбцами: Name и Age.
- Количество строк таблицы настраивается через переменную "number_of_columns".
- Использует случайный выбор из заранее заданных списков имён и возрастов.

## Пример вывода

```text
 фейк данные  
┏━━━━━━┳━━━━━┓
┃ Name ┃ Age ┃
┡━━━━━━╇━━━━━┩
│ вася │  8  │
│ дима │ 15  │
└──────┴─────┘
```

## Установка

Установите библиотеку "rich", если ещё не установлена:

```bash
pip install rich
```

## Запуск 

```bash
python table.py
```
